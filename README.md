# Pagekit Formmaker

Bixies Formmaker 为你的 Pagekit 站点轻松的添加表单。

选择不同的字段类型创建自己的表单，你可以在管理员面板轻松管理表单。

查看演示: [http://formmaker.bixie.org](http://formmaker.bixie.org).

## 安装

你可以通过 Pagekit 后台内置的应用市场安装该插件

![image](http://cdn.guxin.net/bixie.jpg)

其依赖的扩展将会自动安装
